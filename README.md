Финансовый менеджер. Веб-приложение для анализа доходов и расходов.

Функционал

Транзакции (/):
Полный список всех операций.
Фильтрация по типу (доход/расход), категории и дате.
Создание, редактирование и удаление записей (/create/, /<id>/edit/, /<id>/delete/).

Категории (/categories/):
Просмотр списка всех категорий.
Создание новых и удаление существующих категорий.

Статистика (/statistics/):
Графики и сводки за выбранный период.
Анализ распределения средств по категориям.

REST API (/api/):
Отдельно API.

Технологии
Бэкенд: Python, Django

Фронтенд: HTML, CSS, JavaScript

База данных: SQLite
